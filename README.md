# gt-security-vulnerabilities-visualization
## Installation

```st
Metacello new
	repository: 'github://IgnacioMoralesA/gt-security-vulnerabilities-visualization:main/src';
	baseline: 'GtSecurityVulnerabilitiesVisualization';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfGtSecurityVulnerabilitiesVisualization asClass loadLepiter
```
